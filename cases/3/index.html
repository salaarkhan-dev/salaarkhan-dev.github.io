Recorded Difficulties

  • Render real‑time shader art without smudging accessibility or battery
  • Restore tap‑targets after iOS time picker overlays ghosted the UI
  • Skirt autoplay restrictions while delivering gentle audio confirmations
  • Balance a luxurious glassmorphism vibe with strict performance budgets
  • Ship a PWA that behaves — install flow, offline safety, and updates

Remedial Actions

  • GPU‑accelerated shader canvas with throttled animation and responsive density; fallbacks for low‑power contexts
  • iOS input ritual: blur and refocus the panel, force reflow and compositing nudge to dismiss phantom layers after picker close
  • Dual‑track audio: HTML5 Audio for toasts, Web Audio completion chord unlocked on first gesture with resilient resume
  • Tailwind‑driven tokens and layered blur only where needed; careful z‑order and focus rings for keyboard clarity
  • Service worker for offline assets and a polite install prompt; update channel gated behind a small, reversible refresh

Findings & Determinations

  • Tactile interactions with no stuck overlays; 'Set Reminder' stays crisp on iOS

  • Two modes — Simple and Immersive — both responsive and battery‑aware
  • PWA install path that feels native without hijacking the user
